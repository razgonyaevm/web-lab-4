🚀 Web Lab 4

Приложение для проверки попадания точек в область

📋 О приложении

  • Функционал: Интерактивная проверка попадания точек в геометрические области
  • Технологии: Vue.js 3 + Spring WebFlux + PostgreSQL + Redis
  • Безопасность: JWT аутентификация, rate limiting, валидация
  • Адаптивность: Работает на всех устройствах (desktop/tablet/mobile)
💻 Режим разработки: Приложение загружается... Если загрузка занимает слишком много времени, проверьте консоль браузера (F12).
❌ Ошибка загрузки: Не удалось загрузить приложение. Проверьте подключение к интернету и консоль браузера для деталей.
